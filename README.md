## Course Description

This course introduces students to tools and techniques commonly used by security professionals and hackers. Students will gain practical experience with penetration testing, vulnerability assessment, and defensive measures. Upon completion, students will be prepared for CompTIA Security+ or CEH (Certified Ethical Hacker) certification exams.

**Prerequisite:** Hacking Defense Techniques (駭客攻防技術)

## Course Objectives

### Knowledge
- Understand the steps and methodologies of hacker intrusions
- Analyze virus code patterns
- Build foundational knowledge in information and network security
- Learn tools and techniques used by security professionals

### Skills
- Gain hands-on experience with security tools and hacking techniques
- Design and implement defensive mechanisms against attacks

### Attitude
- Develop diligent, dedicated, and cautious learning spirit
- Foster practical problem-solving mindset

4. **Wireless Technologies, Security, and Attacks**
